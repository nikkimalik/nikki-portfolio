# 🌐 Nikki Malik Portfolio

![Portfolio Preview](https://nikkimalik.pages.dev/og-image.png)

A modern and responsive portfolio website built to showcase my projects, skills, experience, and development journey as a **MERN Stack Developer** and **AI Trainer**.

## Live Demo

🔗 **Portfolio Website:** https://nikkimalik.pages.dev

## Features

- Fully responsive design
- Smooth animations with Framer Motion
- Modern dark-themed UI
- Dedicated sections for About, Skills, Projects, Lab, and Contact
- Open Graph (OG) support for social media sharing
- Custom branding and favicon
- Fast performance powered by Vite
- Automatic deployment with Cloudflare Pages

## Tech Stack

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/nikkimalik/nikki-portfolio.git
```

### Navigate to the Project

```bash
cd nikki-portfolio
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment

The portfolio is deployed on **Cloudflare Pages** with automatic CI/CD integration through GitHub.

Every push to the production branch automatically triggers a new deployment.
